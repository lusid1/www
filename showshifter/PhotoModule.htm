Photo Module

This is a Photo Viewer Module for Showshifter.  It support the following features:

  • Folder based Image Browser

  • List View Mode (fast)

  • Thumbnail View Mode (pretty)

  • Folder Previews in Thumbnail View

  • Zoom & Pan images

  • Slideshow Mode

  • Supports more file types than the default photo viewer

  • Compatible with Showshifter 3.10 Final

  • Versions for 3.01 or older are still available for download, but not supported.
