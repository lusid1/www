Modules

Mame Module

This is a MAME front-end implemented as a Showshifter plug-in. It supports recent win32 console versions of MAME, and mame screenshots in PNG format.
Photo Module

This is an alternative Photo Viewer for Showshifter. It supports folders of pictures using a directory browser using either list view or thumbnail view. It also supports image rotation, zoom and pan.
External Player Module

This is an external player module for Showshifter. It lets you open a video file from Showshifter's video read folders, and play it in the application of your choice. It also auto mute's the live-tv, and has mute enforcement to keep showshifter from un-muting it.

It's intended use it to watch an existing recording in another media player while showshifter is busy recording something else. Although it should work with any media player, it works really well with Zoom Player. Zoom Player can be downloaded from www.inmatrix.com.
zoomconfig.zip

This file contains my zoom player settings, and my zoom player key mappings. Several of the keys used in media playback are set to use showshifter's keyboard mappings, and the settings allow for near seemless integration. They can get you up and running fast.
HelloWorld.zip

This is a simple Hello World showshifter module, with source code. It displays a page with some text and an exit button. Not particularly useful, unless you are interested in reading the source.
CommonControls.zip

This module builds upon the hello world module, adding some common controls to the interface. It uses the Edit, CheckBox, Button, and DropList controls as well as the MessageBox and the FilePicker. It can popup a messagebox and launch external programs. Still nothing useful, but the source is included.
NoCD.zip

This is a replacement CD module, that will not show itself on the main menu and will not be accessible. Use it to disable the CD module if you don't want it to be available.
NoDVD.zip

This is a replacement DVD module, that will not show itself on the main menu and will not be accessible. Use it to disable the DVD module if you don't want it to be available.
NoJukebox.zip

This is a replacement Jukebox module, that will not show itself on the main menu and will not be accessible. Use it to disable the Jukebox module if you don't want it to be available.
NoPictureViewer.zip

This is a replacement Picture Viewer module, that will not show itself on the main menu and will not be accessible. Use it to disable the Picture Viewer module if you don't want it to be available.

Scripts

MakeAVS.zip

This script will create AVISynth files for SS AVI formatted recordings. With this you can watch a whole recording as easily as if you had recorded it in SSF format. Requires AVISynth v0.3
showsaver.zip

This script can be used to save showshifted video in either an automated or on-demand mode. Normaly showshifted files are deleted by showshifter, sometimes you might want to keep them.
VdubSSF.zip

This script automates the process of appending ss avi files together in virtualdub.

ssscripts.zip

This is a collection of scripts I wrote to implement a recompression server. They run outside of showshifter (on a network server in my case), and process recordings using VirtualDub and TmpgEnc. They are capable of producing (s)VCD and DVD ready mpeg files from SS AVI recordings. You'll need VirtualDub, TmpgEnc, and Vdubs' frameserver all working together just to get started with these.
