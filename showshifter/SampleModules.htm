Modules

HelloWorld.zip

This is a simple Hello World showshifter module, with source code. It displays a page with some text and an exit button. Not particularly useful, unless you are interested in reading the source.
CommonControls.zip

This module builds upon the hello world module, adding some common controls to the interface. It uses the Edit, CheckBox, Button, and DropList controls as well as the MessageBox and the FilePicker. It can popup a messagebox and launch external programs. Still nothing useful, but the source is included.
NoCD.zip

This is a replacement CD module, that will not show itself on the main menu and will not be accessible. Use it to disable the CD module if you don't want it to be available.
NoDVD.zip

This is a replacement DVD module, that will not show itself on the main menu and will not be accessible. Use it to disable the DVD module if you don't want it to be available.
NoJukebox.zip

This is a replacement Jukebox module, that will not show itself on the main menu and will not be accessible. Use it to disable the Jukebox module if you don't want it to be available.
NoPictureViewer.zip

This is a replacement Picture Viewer module, that will not show itself on the main menu and will not be accessible. Use it to disable the Picture Viewer module if you don't want it to be available.
