Mame Module

This is a Mame Front End plugin for Showshifter.   Features include:

  • Launch Mame games with your remote in Showshifter

  • Set common Mame options defaults, and customize them on a per game basis

  • Uses Mame's standard PNG screenshots

  • Ability to delete screenshots from within Showshifter

  • Ability to hide clone sets

  • Ability to hide unavailable sets

  • Compatable with Showshifter 3.10 Beta

  • Versions for 3.01 or older are still available for download, but not supported.
